How to Export

  1. Open this file in Chrome or Safari
  2. Open DevTools (Cmd+Option+I)
  3. For each size, right-click the .favicon-container element and select "Inspect"
  4. In Chrome: Cmd+Shift+P, type "Capture node screenshot" and press Enter
  5. The browser will save a PNG at the exact pixel dimensions of the container
  6. Rename each export to its target filename (e.g. apple-touch-icon.png)
  7. For favicon.ico, combine the 16x16, 32x32, and 48x48 PNGs using an ICO converter
